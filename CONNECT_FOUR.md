# Connect Four

Simple GUI Connect Four implemented with Tkinter and ttkbootstrap.

## Files
- [`connect_four.py`](/Connect%20Four%20Game/connect_four.py) â€” Main game script.

## Requirements
- Python 3.8+
- ttkbootstrap

Install dependency:
```powershell
pip install ttkbootstrap
```

## Controls
- Click the down-arrow buttons above a column to drop a piece.
- Click "Restart" to reset the board.

## Notes
- Players alternate between "X" (red) and "O" (yellow).
- The game checks for 4-in-a-row horizontally, vertically, and diagonally.
